# 🠖 Analisis Sentimen Opini AI di Perguruan Tinggi

Model NLP untuk mendeteksi sentimen opini terhadap penggunaan AI dalam konteks pendidikan tinggi Indonesia.

## Fitur
- Custom preprocessing untuk slang & abbreviasi Indonesia
- TF-IDF vectorization
- SVM Classification model
- Klasifikasi multi-label: **Positif** | **Netral** | **Negatif**

## Cara Penggunaan
Masukkan opini atau pendapat tentang penggunaan AI di perguruan tinggi, lalu klik **Submit** untuk melihat hasil analisis sentimen.
